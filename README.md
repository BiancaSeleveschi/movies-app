# MovieApp
This movie app offers a seamless experience for movie enthusiasts. With this app, users can easily search for movies based on their preferences. It sorts movies by genre, so you can explore them and also dive into juicy details.
## Features
- view movie list
- search movie
- view movies by genres
- view a movie details
